# Fundamentos de Programación en Python 🐍

¡Bienvenido a mi repositorio de aprendizaje! Aquí documento mi progreso, ejercicios y apuntes sobre los fundamentos de Python. El objetivo de este espacio es organizar el material de clase y practicar la lógica de programación.

## 📂 Estructura del Repositorio

El contenido está organizado por categorías para facilitar la navegación:

1. **[ApuntesClase](./1.ApuntesClase):** Notas teóricas y conceptos clave vistos en las sesiones.
2. **[Informacion](./2.Informacion):** Guías, recursos adicionales y documentación útil.
3. **[Temas](./3.Temas):** Explicación detallada de temas específicos (ciclos, condicionales, etc.).
4. **[Ejercicios](./4.Ejercicios):** Prácticas pequeñas para reforzar la lógica.
5. **[TestPython](./5.TestPython):** Pruebas y evaluaciones de código.
6. **[Talleres](./6.Talleres):** Proyectos y retos más extensos desarrollados en clase.
7. **[JuegosPython](./7.JuegosPython):** Pequeños proyectos lúdicos desarrollados con Python.

## 🛠️ Requisitos
* Python 3.x
* Un editor de código (como VS Code) o GitKraken para la gestión del repositorio.
